# Marker Match

Tool for detecting, matching, and organizing marker palettes for coloring workflows and creative automation.

## Overview
Marker Match helps detect color palettes from source materials and match them with available marker sets.

## Goals
- detect palette blocks
- extract colors
- match colors to marker databases
- support creative workflows

## Features
- palette extraction
- color matching
- OCR-assisted processing
- export for further use

## Stack
- Python
- OCR
- image processing
- CSV / XLSX outputs

## Status
Public project. Prototype / active development.
